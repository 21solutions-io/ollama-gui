# The missing GUI for the ollama

## installation

First you need to download and install [ollama](https://ollama.ai/download).

Then clone the repo and run the GUI.

```shell
git clone git@github.com:HelgeSverre/ollama-gui.git
cd ollama-gui
yarn install
yarn dev
```

Done.
